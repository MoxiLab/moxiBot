# スターボードコマンドとは？

`starboard` コマンドを使うと、サーバー内の優れたメッセージを特別なチャンネルで紹介できます。メッセージが十分なリアクション（例：⭐）を受け取ると、ボットが自動的にスターボードチャンネルに投稿し、みんなに見てもらえます。

## 使い方

1. **スターボードチャンネルの設定：**
   - `/starboard set <チャンネル>` コマンドで、注目メッセージを表示するチャンネルを選択します。

2. **必要なリアクション数の設定：**
   - 何個のリアクション（例：スター）が必要かを設定できます。例：`/starboard threshold 3` なら、3つ以上のスターが付いたメッセージだけが紹介されます。

3. **システムの無効化や調整：**
   - スターボードを無効にするには `/starboard disable` を使います。
   - 同じコマンドでチャンネルやしきい値もいつでも変更できます。

## 使用例

- `/starboard set #注目` → #注目 チャンネルをスターボードに設定。
- `/starboard threshold 5` → 5つ以上のスターが付いたメッセージのみ紹介。
- `/starboard disable` → スターボードシステムを無効化。

## 注意
- 管理者のみがスターボードを設定できます。
- ボットにはスターボードチャンネルでメッセージの閲覧・送信権限が必要です。
- 削除または編集されたメッセージはスターボードに表示されなくなる場合があります。

こうして、コミュニティの質の高い投稿やベストな貢献を称えることができます！
