# स्टारबोर्ड कमांड क्या है?

`starboard` कमांड आपको अपने सर्वर के सर्वश्रेष्ठ संदेशों को एक विशेष चैनल में हाइलाइट करने की सुविधा देता है। जब किसी संदेश को पर्याप्त प्रतिक्रियाएँ (जैसे ⭐) मिलती हैं, तो बॉट उसे starboard चैनल में सभी के लिए पोस्ट कर देता है।

## इसका उपयोग कैसे करें?

1. **starboard चैनल सेट करें:**
   - `/starboard set <चैनल>` कमांड का उपयोग करें ताकि हाइलाइट किए गए संदेश कहाँ दिखेंगे।

2. **आवश्यक प्रतिक्रियाओं की संख्या निर्धारित करें:**
   - आप तय कर सकते हैं कि किसी संदेश को हाइलाइट करने के लिए कितनी प्रतिक्रियाएँ (जैसे स्टार) चाहिए। उदाहरण: `/starboard threshold 3` केवल उन्हीं संदेशों को हाइलाइट करेगा जिन पर 3 या अधिक स्टार हैं।

3. **सिस्टम को निष्क्रिय या समायोजित करें:**
   - starboard को निष्क्रिय करने के लिए `/starboard disable` का उपयोग करें।
   - आप कभी भी वही कमांड इस्तेमाल करके चैनल या थ्रेशोल्ड बदल सकते हैं।

## उपयोग का उदाहरण

- `/starboard set #हाइलाइट्स` → #हाइलाइट्स को starboard चैनल बनाता है।
- `/starboard threshold 5` → केवल 5 या अधिक स्टार वाले संदेश हाइलाइट होंगे।
- `/starboard disable` → starboard सिस्टम को निष्क्रिय करता है।

## नोट्स
- केवल व्यवस्थापक ही starboard को कॉन्फ़िगर कर सकते हैं।
- बॉट को starboard चैनल में संदेश देखने और भेजने की अनुमति होनी चाहिए।
- हटाए गए या संपादित संदेश starboard में दिखाई नहीं दे सकते हैं।

इस तरह आप अपनी कम्युनिटी को गुणवत्तापूर्ण सामग्री साझा करने और सर्वश्रेष्ठ योगदानों को पहचानने के लिए प्रेरित कर सकते हैं!
